# Fit Queue

運動排程 PWA。使用者可以先選好訓練動作，依序播放，並透過節拍、語音提示與示範影片完成訓練。

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Demo Videos

把影片放在 `public/demos/`，例如：

```text
public/demos/squat.mp4
```

後台填入：

```text
/demos/squat.mp4
```

## Admin

本機後台路徑：

```text
/admin
```
